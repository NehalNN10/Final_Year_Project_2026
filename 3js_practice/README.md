# Setup For Three.js
## Install Node.js

Link for Installation : https://nodejs.org/en/download

## Fixing npm execution policy error

Open Powershell as administrator, use the following command : Set-ExecutionPolicy RemoteSigned
and then type in Y.

## Download Live Server extension
Link: https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer

## Run index.html file in Live Server

Right click anywhere on the html file and click on run server, which will run the file on your local browser
